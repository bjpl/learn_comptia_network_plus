**Problem:** `deviceTemplates` defaultSpecs missing required `redundancy` and `hotSwappable` properties

```typescript

  maxConnections: 1000,
  powerConsumption: '50W',
  redundancy: false,
  hotSwappable: false,
}
```
**Files Affected:**
